**1.	Specifications**

- Dimension: 21×21mm.
- Input voltage: 4.5V÷28V.
- Fixed out voltage: 3.3V (You can adjust the voltage by changing feedback resistors).
- Current output: 3A.

**2.	Notations**
- +VIN: Input voltage.
- GND: Common Ground for input and output voltage.
- +3V3: Output voltage.
